### Using the API

URL  

`/mangas/` or `/mangas/<id>/`  

Methods  
GET | POST | PUT | DELETE  

Data Params  
 {  
 	"title": [string],  
 	"image": [string],  
 	"description": [string],  
 	"author": [string],  
 	"genre": [string],  
 	"published": [date],  
 	"rank": [integer]  
 }
